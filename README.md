# Password Strength Analyzer (local, no dependencies)

Open `index.html` in a browser to get an instant strength score, feedback, and rough crack-time estimates.

## Run

- Double-click `index.html`, or
- From PowerShell:
  - `Start-Process .\\index.html`

## Notes

- This is a heuristic analyzer (not a guarantee of security).
- Crack-time estimates are approximate and assume different attacker guess rates.
